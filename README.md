## 五、代码结构

```
Step 1: 数据获取          ← 腾讯理财接口（个股K线 + 沪深300指数）
Step 2: 数据清洗与指标     ← MA、MACD、RSI、对数成交量、滞后变量
Step 3: 绘制走势图        ← K线+均线+MACD（可自行补充RSI、布林带）
Step 4: 多元回归建模       ← OLS估计 + 模型诊断
Step 5: 一步向前预测       ← 用今天信息预测明天（真正可实战的预测）
Step 6: 结论与讨论         ← 学生自行完善
```

## 六、核心知识点

### 6.1 技术指标

| 指标 | 公式 | 含义 |
|------|------|------|
| **MA5/MA20/MA60** | `close.rolling(n).mean()` | n日移动平均线，判断趋势方向 |
| **MACD** | `DIF = EMA12 - EMA26`<br>`DEA = EMA9(DIF)`<br>`MACD柱 = 2*(DIF-DEA)` | 价格动能与趋势转向信号 |
| **RSI(14)** | `100 - 100/(1 + 平均涨幅/平均跌幅)` | 超买(>70)/超卖(<30)判断 |

### 6.2 回归模型

**同期模型（样本内拟合）**

```
close_t = β₀ + β₁·hs300_t + β₂·log_volume_t + β₃·lag_close_t
        + β₄·lag5_ret_t + β₅·MACD_HIST_t + β₆·RSI_t + ε_t
```

**一步向前模型（真正预测）**

```
close_{t+1} = β₀ + β₁·hs300_t + β₂·log_volume_t + β₃·close_t
            + β₄·lag5_ret_t + β₅·MACD_HIST_t + β₆·RSI_t + ε_t
```

> ⚠️ **关键区别**：不能用 `close_t` 同时当 Y 和 X 来"预测自己"。
> 预测未来时，只能使用 **t 期及之前已知的信息**。

### 6.3 模型诊断

| 检验 | 用途 | 判断标准 |
|------|------|---------|
| **DW检验** | 残差自相关 | 1.5 ~ 2.5 为正常 |
| **White检验** | 异方差 | p < 0.05 存在异方差，需用稳健标准误 |
| **VIF** | 多重共线性 | VIF > 10 说明存在严重共线性 |

## 九、注意事项

1. **前复权价格**：本代码使用腾讯接口的 `qfqday` 字段（前复权），避免了除权除息导致的价格跳变，适合用于时间序列分析。
2. **指数K线**：沪深300指数使用 `day` 字段（指数本身无需复权）。
3. **预测局限**：
   - 递归多步预测假设未来市场指标（hs300、MACD、RSI等）不变，实际中这些都会变化
   - 模型基于历史数据，无法预测黑天鹅事件
   - **本代码仅供学习，不构成投资建议**
